# Chess

This is a C++ Terminal Chess implementation

It is still in progress at the moment


To build:
   make

To Run:
   ./chess


It looks like this:


W = White, B = Black
K = King, Q = Queen, B = Bishop, H = Knight, R = Rook, P = Pawn
           A    B    C    D    E    F    G    H  
          ----------------------------------------
        1 | RW | PW |    |    |    |    | PB | RB |
          ----------------------------------------
        2 | HW | PW |    |    |    |    | PB | HB |
          ----------------------------------------
        3 | BW | PW |    |    |    |    | PB | BB |
          ----------------------------------------
        4 | QW | PW |    |    |    |    | PB | KB |
          ----------------------------------------
  white 5 | KW | PW |    |    |    |    | PB | QB |  black  
          ----------------------------------------
        6 | BW | PW |    |    |    |    | PB | BB |
          ----------------------------------------
        7 | HW | PW |    |    |    |    | PB | HB |
          ----------------------------------------
        8 | RW | PW |    |    |    |    | PB | RB |
          ----------------------------------------
white's turn!
